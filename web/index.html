Bit Command Cheat Sheet

                
Commands run from top to bottom. Remember the parentheses: ().

            

            
        


        

            

                
Move

                

                    
bit.move()

                    
Move forward one square.

                    
bit.turn_left()

                    
Turn left without moving.

                    
bit.turn_right()

                    
Turn right without moving.

                

            


            

                
Color

                

                    
bit.paint("blue")

                    
Paint Bit's square. Change "blue" to the color you want.

                    
bit.erase()

                    
Make Bit's square white.

                    
bit.get_color()

                    
Find the color of Bit's square.

                

                
Colors: white, black, orange, green, yellow, blue, red, purple

            



            

                
Look

                
These answer with True for yes or False for no.

                

                    
bit.can_move_front()

                    
Check whether Bit can move forward.

                    
bit.can_move_left()

                    
Check whether the square to the left is open.

                    
bit.can_move_right()

                    
Check whether the square to the right is open.

                

            


            

                
Check a Color

                

                    
bit.is_on_blue()

                    
Check whether Bit is on blue.

                    
bit.is_on_green()

                    
Check whether Bit is on green.

                    
bit.is_on_red()

                    
Check whether Bit is on red.

                    
bit.is_on_white()

                    
Check whether Bit is on an empty white square.

                

            


            

                
Example

                
if bit.can_move_front():
    bit.move()
    bit.paint("blue")

                
The indented commands run only when Bit can move forward.
